🏥 Projet Gestion Séjour

Un système de Gestion de Séjour d'un Hopital

Contexte: Projet d'Atelier Professionnel fait en 2e Année de BTS SIO, autour d'un hôpital souhaitant une application afin de gérer les séjours des patients

But: Développer une application de gestion des séjours pour suivre l'hospitalisation des patients (intervention, examen, ambulatoire, lit et chambre)

Technologies utilisées: Symfony (Framework), PHP, HTML, CSS

Base de données: Postgres

Projet: Projet_Gestion_Patient

Application complète de gestion des séjours des patients en milieu hospitalier.

Fonctionnalités principales :

Gestion des séjours selon le rôle :

• Personnel Administratif: Ajouter un nouveau patient, modifier des patients existants

• Personnel Administratif: Gérer les séjours des patients

• Infirmier: Prendre en compte l'arrivée et la sortie d'un patient dans un service

• Infirmier: Consulter les séjours à une date donnée

• Infirmier: Consulter les séjours commençant à une date donnée

• Infirmier: Consulter les séjours à venir

Versions du projet :

• Une version faite lors du travail de groupe qui est incomplète et non fonctionnelle

• Une version faite seul après la deadline du travail et qui est complète

Ce dépôt correspond à la version faite seul.

Fonctionnalité complémentaire :

• Possibilité de se connecter avec un identifiant et un mot de passe

🚗 Projet Réservation de Véhicule

Un système de Réservation de Véhicule

Contexte: Projet d'Atelier Professionnel fait en 2e Année de BTS SIO, autour d'une entreprise souhaitant une application afin de gérer la réservation de ses véhicules

But: Développer une application de gestion des réservations de véhicules pour suivre les demandes et les disponibilités

Technologies utilisées: Java

Base de données: Postgres

Projet: Projet_Reservation_Vehicule

Application complète de gestion des réservations de véhicules pour une entreprise.

Fonctionnalités principales :

Gestion des réservations selon le rôle :

• Employé: Faire une demande de réservation pour un véhicule à une date donnée

• Employé: Consulter ses demandes de réservation passées et à venir

• Gestionnaire: Consulter toutes les demandes de réservation

• Gestionnaire: Accepter ou refuser une demande de réservation
